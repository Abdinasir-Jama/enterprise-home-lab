# Enterprise Home Lab

## Overview

Hello, I made this repository to document my home lab that was designed to develop my pratical understanding and skills in networking and system adminstration. 
This lab is hosted on arch linux using QEMU/KVM and it will continue to grow and evolve as i implement new configurations and services.

---

## Objectives

- Design and manage an enterprise-style virtual network
- Develop practical networking skills
- Deploy and administer Windows Server environments
- Learn Active Directory, DNS, DHCP and Group Policy
- Improve Linux system administration skills
- Gain experience with network security and firewall configuration
- Build automation using PowerShell and Python
- Document projects using professional engineering practices
